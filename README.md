## Getting Started

To get a local copy up and running follow these simple example steps.

### Install
Have NodeJS installed and follow the next steps
### Usage
- `npm i` for installing dev dependecies
- `npm run build` or `npm start` to see the project
### Run tests
- `npm test`
### Deployment
- `npm run deploy`
